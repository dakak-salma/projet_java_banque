
Projet 3 : Système de Gestion Bancaire

Objectifs du projet Ce projet vise à développer une application de gestion bancaire permettant de gérer différents types de comptes, d’effectuer des opérations bancaires et de conserver l’historique des transactions.

Les objectifs pédagogiques sont :

Maîtriser le polymorphisme et l’abstraction
Implémenter des interfaces
Gérer les transactions avec des collections
Appliquer des patterns de conception (Factory)
Gérer des exceptions métier personnalisées
Technologies utilisées

Java 8+
Programmation Orientée Objet
Collections Framework
API Date/Time (LocalDateTime)
UML
GitHub Instructions d’exécution
Ouvrir le projet dans IntelliJ IDEA
Compiler le projet
Lancer la classe Main

Gestion des clients
